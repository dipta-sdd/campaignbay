Woocommerce.php : 41

Woocommerce Class.

This helper class provides a set of static methods to abstract and simplify common interactions with the WooCommerce plugin. It serves as a central utility for retrieving product data, getting order information, formatting prices, and handling other WooCommerce-specific tasks in a consistent manner throughout the plugin.
