Filter.php : 37

Filter Class.

This helper class provides methods to evaluate whether a given WooCommerce product matches the targeting conditions of a CampaignBay campaign. It handles logic for product, category, and sale status matching, including exclusion rules.

It is designed as a singleton to ensure a single, consistent instance throughout a request.
